#  Estacionamento Console

Projeto desenvolvido com foco em Domain-Driven Design (DDD), utilizando .NET e arquitetura orientada a domínio.

##  Tecnologias
- .NET 9
- GitHub Actions (CI/CD)
- Docker
- Docker Hub

## ⚙️ Pipeline CI/CD

O projeto possui pipeline automatizado com:

- Build automático
- Execução de testes
- Criação de imagem Docker
- Publicação no Docker Hub

## 🐳 Executar com Docker

```bash
docker run --rm -it annahermenegildo/estacionamentoconsole
